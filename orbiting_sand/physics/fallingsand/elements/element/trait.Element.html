This gets the last time the element was processed Useful for physics calculations by getting the dt between now and then

source

fn get_color(&self) -> Color

This gets the color of the element Always constant and unique for each element, so that we can process them in fragment shaders knowing their type just by their color You can map them to other colors and add effects using the fragment shader

source

fn get_density(&self) -> Density

This gets the density of the element relative to the cell_width This is so bigger cells have more mass, so we don’t have to have as many cells for simpler bodies, like gas giants or the sun
