Expand description

An element grid is a 2D grid of elements tied to a chunk

Fields§

§grid: Grid<Box<dyn Element>>§coords: ChunkCoords§total_mass: Mass

Some low resolution data about the world

§already_processed: bool

The minimum temperature in the chunk that is not 0 The maximum temperature in the chunk This deals with a lock during convolution

§last_set: Clock

This deals with whether or not the element grid needs to be processed or if it hasn’t seen any changes since the last frame maybe you can skip it
